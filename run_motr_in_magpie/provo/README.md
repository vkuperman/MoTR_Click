# MoTR Provo

Mouse-tracking for Reading — Provo variant (three lists, SONA, McMaster ethics).

## How to run

```bash
cd run_motr_in_magpie/provo
npm install
npm run serve
```

Open the URL shown in the terminal (e.g. http://localhost:8080).

## Build for production

```bash
npm run build
```

Output is in `dist/`.
